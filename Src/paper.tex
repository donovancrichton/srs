  \subsection*{Syntax}
  \subsubsection{$p$}
  $p$ represents the message preamble $0x505a$ followed
  by a 16-bit length in 32-bit words, followed by the message
  type 2-word block string (e.g.) ``hello''.
  \subsubsection{$v$}
  $v$ represents the 4-character long string containing the
  version of the ZRTP protocol.
  \subsubsection{$cid$}
  $cid$ represents the client identificationn string which
  identifies the vendor and release of the ZRTP software.
  \subsubsection($h3$)
  $h3$ represents the following: \\
    $h0$ representing a 256-bit random nonce. \\
    $h1$ representing a hash of $h0$ with SHA-256.\\
    $h2$ representing a hash of $h1$ with SHA-256.\\
